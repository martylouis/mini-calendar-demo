# BA Mini Calendar

## How to Use

Clone locally and install packages with  [npm](https://docs.npmjs.com/cli/init) or [Yarn](https://yarnpkg.com/lang/en/docs/cli/create/)

```sh
npm install
# or
yarn
```

Run production build

```bash
npm run build
npm run start
#or
yarn build
yarn start
```

Run dev build

```bash
npm run dev
# or
yarn dev
```
